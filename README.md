# Creative Agency Landing Page

Landing page de uma agência criativa fictícia, desenvolvida para praticar HTML5, CSS3, Grid, animações e responsividade.

## Tecnologias
- HTML5
- CSS3
- Git & GitHub

## Funcionalidades
- Layout com CSS Grid
- Hover animations
- Design responsivo
- Estrutura semântica

## Objetivo
Praticar fundamentos de front-end e versionamento de código.
